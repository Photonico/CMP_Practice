# VASP note

## Prepration

+ Install VASP, add environment variable: `export PATH="~/VASP544/bin:$PATH"`  
+ Install Python package `py4vasp`: `pip install py4vasp`  

## Input files

+ INCAR
    + The calculation information

+ KPOINTS
    + K-points information

+ POSCAR
    + Check the file with Vesta

+ POTCAR
    + Contains the pseudopotential for each atomic species used in the calculation

## Run code

+ With the package `py4vasp`
